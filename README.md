# 🎀 Hannah Lee Website

A modern personal portfolio website built with Next.js to showcase my software engineering projects, technical skills, and experience. Inspired by Pinterest scrapbook and corkboard aesthetics, the site combines handcrafted visuals, playful interactions, and responsive design to create a portfolio that reflects both my personality and my work.

## Features

- **📌 Scrapbook-Inspired Design**
  - Features a custom scrapbook-inspired interface with layered textures, decorative elements, and a Pinterest-inspired corkboard aesthetic.

- **💻 Project Showcase**
  - Browse featured software engineering and projects with descriptions, technologies used, and links to source code or live demos.

- **👩🏻‍💻 About Me**
  - Learn more about my background, experience, technical interests, and career journey in software engineering.

- **🛠️ Skills & Technologies**
  - Displays the programming languages, frameworks, tools, and technologies I use across web, mobile, and software development.

- **📬 Contact**
  - Easily connect through email, GitHub, or LinkedIn directly from the portfolio.

## Prerequisites

- **Node.js 18+**
- **npm** (or yarn/pnpm)
- A modern web browser

---

## Setup

### 1. Clone the repo

```bash
git clone https://github.com/hannahlee4786/HannahLee.git
cd HannahLee
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

The website will be available at:

```
http://localhost:3000
```

## Running the App

1. Clone the repository.
2. Install project dependencies using `npm install`.
3. Start the development server with `npm run dev`.
4. Open **http://localhost:3000** in your browser.
5. Any changes made to the codebase will automatically reload the application.

## Tech Stack

- **Next.js** — React framework powering the application
- **React** — Component-based frontend library
- **TypeScript** — Type-safe JavaScript development
- **HTML** — Semantic page structure and content
- **CSS** — Custom styling, layouts, and responsive design
- **Framer Motion** — Smooth animations and page transitions
